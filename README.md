Это проект для UI-тестирования космической бургерной «Stellar Burgers».
Логика данного тестирования проверяется на языке Python и тестируется Pytest - тестовый фреймворком, который запускает все эти файлы по очереди и указывает сколько тестов прошли проверку успешно, а сколько упало с ошибкой.
Установите зависимости командой: pip install -r requirements.txt
Команда для запуска всех тестов сразу: pytest -v; Команда для запуска конректного теста: pytest -v tests/test_login.py
